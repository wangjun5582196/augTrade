# 🔄 需要重新编译项目

## 问题
新添加的回测功能类文件还没有被编译到 target/classes 目录。

## 解决方案

### 在 IntelliJ IDEA 中：

1. **方法1：重新构建项目（推荐）**
   - 点击菜单：`Build` → `Rebuild Project`
   - 等待编译完成
   - 应用会自动热重载新的类

2. **方法2：停止并重新运行**
   - 点击红色停止按钮停止应用
   - 点击绿色运行按钮重新启动
   - 新功能将被加载

3. **方法3：手动编译（快捷键）**
   - macOS: `Cmd + F9`
   - Windows/Linux: `Ctrl + F9`
   - 等待编译完成

## 验证步骤

编译完成后：

1. 访问：`http://localhost:3131/api/backtest`（应该看到回测页面）
2. 或访问：`http://localhost:3131/backtest.html`
3. 配置回测参数并执行

## 如果还是不工作

请查看 IntelliJ IDEA 的控制台输出，检查是否有以下问题：
- 编译错误
- 类路径问题
- 端口占用
- 数据库连接问题
