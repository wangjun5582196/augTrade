```java
// ✅ 震荡市过滤：检查ADX（平均趋向指标）
Double adx = context.getIndicatorAsDouble("ADX");
if (adx != null && adx < 20) {
    log.warn("[{}] ⚠️ 震荡市检测（ADX={}），EMA信号不可靠，暂不交易", STRATEGY_NAME, adx);
    return createHoldSignal(String.format("震荡市（ADX=%.1f<20），EMA频繁交叉不可靠", adx));
}
```

**效果**：
- ADX < 20时，TrendFilter不发出任何信号
- 震荡市中，EMA频繁金叉死叉产生的虚假信号被屏蔽
- 只在趋势市（ADX ≥ 20）才使用EMA策略

**改进前问题**：
```
震荡市（ADX<20）
→ EMA频繁交叉（金叉→死叉→金叉...）
→ TrendFilter不断改变信号（做多→做空→做多...）
→ 频繁开仓+频繁反转
→ 今天15笔交易，11笔信号反转
```

**改进后预期**：
```
震荡市（ADX<20）
→ TrendFilter返回HOLD信号
→ 其他策略得分不足15分
→ 不开仓
→ 避免今天所有15笔交易
```

## 🎓 改进原理说明

### ADX（平均趋向指标）

**ADX数值含义**：
- **ADX < 20**: 震荡市，无明确趋势
- **ADX 20-25**: 弱趋势开始形成
- **ADX 25-50**: 强趋势
- **ADX > 50**: 极强趋势

**为什么ADX<20不适合EMA策略**：
1. 震荡市中，价格在支撑阻力间反复
2. EMA线会频繁交叉（金叉→死叉→金叉...）
3. 每次交叉都产生信号 → 频繁开仓
4. 但价格没有真实突破 → 容易被打回来
5. **结果**：频繁止损或反转

### 趋势市 vs 震荡市策略

| 市场类型 | ADX | 适合策略 | 不适合策略 |
|---------|-----|----------|-----------|
| **震荡市** | <20 | RSI超买超卖、布林带均值回归 | ❌ EMA趋势跟踪 |
| **弱趋势** | 20-25 | 谨慎使用趋势策略 | 激进策略 |
| **强趋势** | 25-50 | ✅ EMA趋势跟踪、动量突破 | 均值回归 |
| **极强趋势** | >50 | 趋势加仓、移动止盈 | 逆势交易 |
