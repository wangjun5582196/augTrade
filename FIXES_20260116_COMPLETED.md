### 修复 #2: 优化Williams评分区间

**文件**: `BalancedAggressiveStrategy.java`  
**位置**: 第166-187行  
**问题**: Williams -65.74（相当超卖）只得3分，缺少-65这个关键档位

**修复前**:
```java
if (williamsR < -70) {
    buyScore += 5;  // 深度超卖
} else if (williamsR < -60) {
    buyScore += 3;  // 中度超卖
}
```

**修复后**:
```java
if (williamsR < -70) {
    buyScore += 5;  // 深度超卖
} else if (williamsR < -65) {
    buyScore += 4;  // 次深度超卖（新增）
} else if (williamsR < -60) {
    buyScore += 3;  // 中度超卖
}
// 做空方向也同样优化
else if (williamsR > -35) {
    sellScore += 4;  // 次超买（新增）
}
```

**理由**:
- Williams -65.74 已经很接近-70的深度超卖
- 增加-65档位，给予4分（介于5分和3分之间）
- 提高超卖/超买区域的评分权重

**影响**:
- Williams -65.74 从3分提升到4分
- 总评分从6分提升到7分
- **在当前市场条件下可以成功开单** ✅

## 🟡 P1修复 - 重要优化

### 修复 #3: ADX渐进式阈值

**文件**: `BalancedAggressiveStrategy.java`  
**位置**: 第119-133行  
**问题**: ADX 14.99和15.01之间只差0.02，但阈值差3分（12分→9分）

**修复前**:
```java
if (adx < 15) {
    requiredScore = 12;  // 极弱趋势
} else if (adx < 20) {
    requiredScore = 9;   // 震荡市场
}
```

**修复后**:
```java
if (adx < 15) {
    requiredScore = 12;  // 极弱趋势
} else if (adx < 17) {
    requiredScore = 7;   // 弱趋势初期（15-17）
} else if (adx < 20) {
    requiredScore = 8;   // 弱趋势后期（17-20）
}
```

**理由**:
- 避免临界值突变造成的不合理跳变
- 使用渐进式阈值，平滑过渡
- ADX 15.33 现在使用7分阈值（15-17区间）

**影响**:
- ADX 15-17: 阈值从9分降低到7分
- ADX 17-20: 阈值从9分降低到8分
- 更合理的区间划分

### 修复 #4: 降低趋势方向惩罚

**文件**: `BalancedAggressiveStrategy.java`  
**位置**: 第155-167行  
**问题**: ADX>25时逆势交易惩罚+5分过于严厉，理论最高分都达不到

**修复前**:
```java
if (adx > 25) {
    if (trendDirection == ADXCalculator.TrendDirection.DOWN) {
        requiredScore = requiredScore + 5;  // 下降趋势，做多需要+5分
    } else if (trendDirection == ADXCalculator.TrendDirection.UP) {
        requiredScore = requiredScore + 5;  // 上升趋势，做空需要+5分
    }
}
```

**修复后**:
```java
if (adx > 25) {
    if (trendDirection == ADXCalculator.TrendDirection.DOWN) {
        requiredScore = requiredScore + 3;  // 从+5降为+3
    } else if (trendDirection == ADXCalculator.TrendDirection.UP) {
        requiredScore = requiredScore + 3;  // 从+5降为+3
    }
}
```

**理由**:
- +5分惩罚导致阈值可能高达12-14分
- 评分系统理论最高分只有14分（有强K线形态）
- 降低到+3分，仍然保持对逆势交易的谨慎

**影响**:
- 逆势交易门槛从+5分降低到+3分
- ADX 15.33 < 25，此规则当前未触发
- 为未来ADX上升时预留合理的交易空间
