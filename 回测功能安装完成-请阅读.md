### 第2步：创建数据库表

**在你的数据库客户端中执行** `create_backtest_tables.sql` 文件

或者手动复制执行以下SQL：

```sql
USE test;

CREATE TABLE IF NOT EXISTS t_backtest_result (
    id BIGINT AUTO_INCREMENT PRIMARY KEY,
    backtest_id VARCHAR(50) NOT NULL UNIQUE,
    symbol VARCHAR(20) NOT NULL,
    strategy_name VARCHAR(50) NOT NULL,
    `interval` VARCHAR(10) NOT NULL,
    start_time DATETIME NOT NULL,
    end_time DATETIME NOT NULL,
    initial_capital DECIMAL(18, 2) NOT NULL,
    final_capital DECIMAL(18, 2),
    total_profit DECIMAL(18, 2),
    return_rate DECIMAL(10, 4),
    total_trades INT DEFAULT 0,
    profitable_trades INT DEFAULT 0,
    losing_trades INT DEFAULT 0,
    win_rate DECIMAL(10, 4),
    max_profit DECIMAL(18, 2),
    max_loss DECIMAL(18, 2),
    max_drawdown DECIMAL(18, 2),
    max_drawdown_rate DECIMAL(10, 4),
    avg_profit DECIMAL(18, 2),
    avg_loss DECIMAL(18, 2),
    profit_loss_ratio DECIMAL(10, 2),
    sharpe_ratio DECIMAL(10, 2),
    total_fee DECIMAL(18, 2) DEFAULT 0,
    status VARCHAR(20) NOT NULL,
    remark VARCHAR(500),
    create_time DATETIME DEFAULT CURRENT_TIMESTAMP,
    update_time DATETIME DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,
    INDEX idx_symbol (symbol),
    INDEX idx_strategy (strategy_name),
    INDEX idx_status (status),
    INDEX idx_create_time (create_time)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4;

CREATE TABLE IF NOT EXISTS t_backtest_trade (
    id BIGINT AUTO_INCREMENT PRIMARY KEY,
    backtest_id VARCHAR(50) NOT NULL,
    symbol VARCHAR(20) NOT NULL,
    side VARCHAR(10) NOT NULL,
    entry_price DECIMAL(18, 2) NOT NULL,
    entry_time DATETIME NOT NULL,
    exit_price DECIMAL(18, 2),
    exit_time DATETIME,
    quantity DECIMAL(18, 4) NOT NULL,
    profit_loss DECIMAL(18, 2),
    profit_loss_rate DECIMAL(10, 4),
    fee DECIMAL(18, 2) DEFAULT 0,
    holding_minutes INT,
    take_profit_price DECIMAL(18, 2),
    stop_loss_price DECIMAL(18, 2),
    exit_reason VARCHAR(50),
    signal_description VARCHAR(255),
    create_time DATETIME DEFAULT CURRENT_TIMESTAMP,
    INDEX idx_backtest_id (backtest_id),
    INDEX idx_symbol (symbol),
    INDEX idx_entry_time (entry_time),
    FOREIGN KEY (backtest_id) REFERENCES t_backtest_result(backtest_id) ON DELETE CASCADE
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4;
```

## 📊 功能特性

- ✅ **3种策略**：SHORT_TERM（短线趋势）、BREAKOUT（突破）、RSI（超买超卖）
- ✅ **完整模拟**：开仓、止盈(2%)、止损(1%)、平仓
- ✅ **丰富指标**：收益率、胜率、盈亏比、夏普比率、最大回撤
- ✅ **手续费计算**：0.05%（开仓+平仓）
- ✅ **详细记录**：每笔交易的完整信息
- ✅ **可视化展示**：统计卡片 + 交易明细表格
