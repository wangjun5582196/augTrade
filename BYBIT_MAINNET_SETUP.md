### 步骤3：获取API密钥（2分钟）

```
1. 登录Bybit正式网
2. 右上角头像 → 账户与安全 → API管理
3. 点击"创建新密钥"
4. 权限设置：

5. 设置IP白名单（可选，更安全）
6. 记录API Key和Secret
```

⚠️ **API Secret只显示一次，务必保存！**

示例：
```
API Key: aBcD1234EfGh5678

```

### 步骤4：配置项目（1分钟）

编辑 `src/main/resources/application.yml`：

```yaml
bybit:
  api:
    enabled: true
    key: aBcD1234EfGh5678              # ✅ 粘贴你的Key
    secret: XyZ9876543210aBcDeFgHiJkLmN  # ✅ 粘贴你的Secret
    testnet: false                     # ✅ 确认是false
  gold:
    symbol: XAUUSDT
    min-qty: 0.001                     # 最小0.001盎司
    leverage: 2                        # 2倍杠杆
  risk:
    stop-loss-dollars: 5               # 止损$5
    take-profit-dollars: 10            # 止盈$10
```

## ⚠️ 风险控制

### 必须遵守的规则：

1. **每日亏损上限**
   - 连续亏损3单 → 暂停交易
   - 当日亏损超$20 → 暂停交易
   - 第二天重新评估

2. **持续监控**
   - 前3天每天至少检查3次
   - 观察信号质量和成功率
   - 记录每次交易

3. **策略优化**
   - 如果信号过多 → 提高ML阈值
   - 如果亏损频繁 → 降低杠杆
   - 如果盈利稳定 → 可小幅增加手数

4. **心态管理**
   - 不要被短期亏损影响
   - 不要因为盈利就激进
   - 量化交易是概率游戏
