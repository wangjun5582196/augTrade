## 🎯 利润最大化策略

### 策略1：移动止损 (Trailing Stop) ⭐推荐⭐

**原理：**
移动止损能够在价格朝有利方向移动时，自动调整止损位，锁定利润的同时保留上涨空间。

**优势：**
- ✅ 保护已有利润（$41）
- ✅ 保留上涨空间（到$100）
- ✅ 自动跟踪价格
- ✅ 避免过早止盈

**实施方案：**
```
当前盈利：$41
触发条件：盈利 >= $30 时启动移动止损

移动规则：
- 初始止损：锁定 70% 利润 = $41 * 0.7 = $28.7
- 跟踪距离：$8-10（根据ATR调整）
- 当价格上涨时，止损价随之上升
- 当价格回落触及止损价时，自动平仓

示例：
入场价：$2600
当前价：$2641（盈利$41）
移动止损价：$2641 - $10 = $2631（锁定$31利润）

如果价格涨到 $2680（盈利$80）：
移动止损价：$2680 - $10 = $2670（锁定$70利润）
```

## ⚙️ 技术实现要点

### 移动止损算法伪代码

```java
// 每次价格更新时执行
void updateTrailingStop(Position position, BigDecimal currentPrice) {
    BigDecimal unrealizedPnl = calculatePnL(position, currentPrice);
    
    // 触发条件：盈利 >= $30
    if (unrealizedPnl.compareTo(new BigDecimal("30")) >= 0) {
        // 计算新的移动止损价
        BigDecimal trailingDistance = new BigDecimal("10");
        BigDecimal newStopLoss;
        
        if ("LONG".equals(position.getDirection())) {
            newStopLoss = currentPrice.subtract(trailingDistance);
        } else {
            newStopLoss = currentPrice.add(trailingDistance);
        }
        
        // 只能向有利方向移动
        if (isStopLossImproved(position, newStopLoss)) {
            position.setStopLossPrice(newStopLoss);
            position.setTrailingStopEnabled(true);
            log.info("移动止损更新：新止损价 = {}", newStopLoss);
        }
    }
}
```

## 🎓 关键要点总结

1. **移动止损是核心**
   - 保护利润 + 保留空间
   - 自动跟踪，无需盯盘

2. **止损只能上调，不能下调**
   - 多头：止损价只能上升
   - 空头：止损价只能下降

3. **跟踪距离要合理**
   - 太紧：容易被震出
   - 太松：保护不足
   - 建议：根据ATR动态调整

4. **耐心持仓**
   - 趋势未破，持有不动
   - 移动止损会自动保护

5. **避免贪婪**
   - $80-100 利润已是优秀表现
   - 不要期待每次都达到最大值
