```java
boolean isDowntrend = ema20.compareTo(ema50) < 0;
```

```java
int signalStrength = 0;

// ADX得分
if (adx > 30) signalStrength += 50;
else if (adx > 25) signalStrength += 40;
else if (adx > 20) signalStrength += 30;
else signalStrength += 0;  // ADX<20不给分

// 趋势方向得分
if (signal == BUY && isUptrend) signalStrength += 30;  // 顺势
else if (signal == SELL && isDowntrend) signalStrength += 30;
else signalStrength -= 30;  // 逆势扣分！

// Williams得分
if (signal == BUY && williamsR < -70) signalStrength += 20;
else if (signal == SELL && williamsR > -30) signalStrength += 20;

// 最低阈值：信号强度<50不交易
if (signalStrength < 50) {
    return Signal.HOLD;
}
```

## 🎓 十一、核心教训

### ❌ 失败教训

1. **趋势方向比趋势强度更重要**
   - ADX高只说明趋势强，不说明方向
   - 在强趋势上涨中做空 = 送钱
   - 必须先判断方向，再看强度

2. **Williams在强趋势中完全失效**
   - 超买可以持续很久
   - 不能仅凭Williams超买就做空
   - 强趋势中应忽略超买/超卖信号

3. **震荡市要休息**
   - ADX < 15说明无方向
   - 频繁试错累积亏损
   - 宁可错过，不可做错

4. **止损保护不力 = 灾难**
   - 理论止损vs实际亏损差异巨大
   - 需要增加最大亏损强制平仓
   - 需要检查订单执行机制

5. **信号强度计算要简单有效**
   - 复杂的计算反而容易出错
   - ADX是最关键的指标，应占主导
   - 信号强度与交易结果要有强相关性

1. **唯一止盈单的启示**

   - 应该避开高波动期
