## 七、实际使用场景

### 场景1：系统开仓 ✅
```
系统检测到交易信号 → PaperTradingService.openPosition()
                  → 保存到数据库
                  → 🔔 发送飞书开仓通知
                  → 继续执行
```

### 场景2：触发止盈 ✅
```
价格监控检测到止盈 → PaperTradingService.updatePositions()
                  → closePosition(..., "TAKE_PROFIT")
                  → 🔔 发送飞书止盈通知
                  → 更新统计数据
```

### 场景3：信号反转 ✅
```
策略信号改变 → TradingScheduler检测到反转
            → PaperTradingService.closePositionBySignalReversal()
            → 🔔 发送飞书信号反转通知
            → 进入冷却期
```
