# 📊 获取历史数据 - 重要说明

## ⚠️ 重要提示

由于你要求获取2025年6月-12月的数据，但现在是2026年1月，那个时间段已经是历史数据。

**Bybit API 历史数据限制：**
- 免费API通常只能获取**最近几个月**的历史数据
- 更早期的数据可能需要购买历史数据服务
- 建议先获取**最近7-30天**的数据进行回测验证

## 🚀 推荐使用方式

### 方法1：获取最近7天数据（推荐用于测试）

```bash
curl -X POST "http://localhost:3131/api/data/fetch-recent-gold"
```

**预期数据量：**
- 7天 × 24小时 × 12根/小时 = 约 **2,016根K线**
- 获取时间：约 **10-20秒**

### 方法2：获取最近30天数据（推荐用于正式回测）

```bash
curl -X POST "http://localhost:3131/api/data/fetch-month-gold"
```

**预期数据量：**
- 30天 × 24小时 × 12根/小时 = 约 **8,640根K线**
- 获取时间：约 **30-60秒**

### 方法3：自定义时间范围

```bash
curl -X POST "http://localhost:3131/api/data/fetch-historical" \
  -d "symbol=XAUUSD" \
  -d "interval=5" \
  -d "startDate=2026-01-01" \
  -d "endDate=2026-01-09"
```

## 📝 使用步骤

### 步骤1：重新构建项目 ⭐
```
IntelliJ IDEA: Build → Rebuild Project
```

### 步骤2：调用API获取数据
```bash

# 或获取最近30天数据（更完整的回测）
```


### 步骤4：验证数据
```sql
SELECT 
    symbol,
    `interval`,
    MIN(timestamp) as earliest,
    MAX(timestamp) as latest,
    COUNT(*) as count
FROM t_kline 
WHERE symbol = 'XAUUSD' 
AND `interval` = '5'
GROUP BY symbol, `interval`;
```

### 步骤5：开始回测
访问：`http://localhost:3131/api/backtest`
