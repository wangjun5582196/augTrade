# 币安XAUUSDT历史数据抓取指南

## 概述

本指南说明如何使用系统抓取币安XAUUSDT（黄金永续合约）的历史K线数据，用于回测和策略优化。

## 🎯 功能特点

- ✅ 支持抓取最近365天（一年）的历史数据
- ✅ 支持抓取最近N天的数据
- ✅ 支持自定义时间范围抓取
- ✅ 支持多种K线周期（1m, 5m, 1h, 1d等）
- ✅ 自动批量抓取（币安每次最多1500条）
- ✅ 自动去重（重复数据会被跳过）
- ✅ 进度实时显示
- ✅ 数据自动保存到数据库

## 🚀 使用方法

### 方法1：使用Shell脚本（推荐）

#### 1. 启动应用

```bash
mvn spring-boot:run
```

或

```bash
java -jar target/aug-trade-*.jar
```

#### 2. 运行抓取脚本

```bash
# 抓取一年数据（5分钟K线）
./fetch_binance_data.sh one-year

# 抓取一年数据（1小时K线）
./fetch_binance_data.sh one-year 1h

# 抓取最近30天数据（5分钟K线）
./fetch_binance_data.sh recent 30

# 抓取最近7天数据（15分钟K线）
./fetch_binance_data.sh recent 7 15m

# 抓取2025年全年数据（5分钟K线）
./fetch_binance_data.sh custom 2025-01-01 2025-12-31

# 抓取2025年全年数据（1小时K线）
./fetch_binance_data.sh custom 2025-01-01 2025-12-31 1h
```

### 方法2：使用HTTP API


## 💾 数据存储

抓取的数据会自动保存到数据库中：

- **表名**: `kline`
- **字段**:
  - `symbol`: XAUUSDT
  - `interval`: K线周期（如 5m, 1h）
  - `timestamp`: 开盘时间
  - `open_price`: 开盘价
  - `high_price`: 最高价
  - `low_price`: 最低价
  - `close_price`: 收盘价
  - `volume`: 成交量
  - `amount`: 成交额

查询示例：

```sql
-- 查看已抓取的数据量
SELECT interval, COUNT(*) as count, 
       MIN(timestamp) as start_time, 
       MAX(timestamp) as end_time
FROM kline 
WHERE symbol = 'XAUUSDT'
GROUP BY interval;

-- 查看最新的10条5分钟K线
SELECT * FROM kline 
WHERE symbol = 'XAUUSDT' AND interval = '5m'
ORDER BY timestamp DESC 
LIMIT 10;
```

## 🔧 常见问题

### Q1: 抓取时提示"服务未启动"

**A:** 请先启动应用：
```bash
mvn spring-boot:run
```

### Q2: 抓取速度很慢

**A:** 
- 检查网络连接
- 币安API有速率限制（1200请求/分钟）
- 可以使用更大的K线周期（如1h代替5m）

### Q3: 如何验证数据已成功保存？

**A:** 查询数据库：
```sql
SELECT COUNT(*) FROM kline WHERE symbol = 'XAUUSDT' AND interval = '5m';
```

或查看应用日志：
```bash
tail -f logs/aug-trade.log
```

### Q4: 重复运行会重复保存数据吗？

**A:** 不会。系统会自动跳过重复的数据，只保存新数据。

### Q5: 可以抓取其他交易对的数据吗？

**A:** 当前版本只支持XAUUSDT。如需其他交易对，请修改控制器代码。

### Q6: 数据抓取失败怎么办？

**A:** 
1. 检查网络连接
2. 查看应用日志获取详细错误信息
3. 确认币安API是否可访问：`curl https://fapi.binance.com/fapi/v1/ping`
4. 重试抓取（重复数据会自动跳过）


   ```bash
   ./fetch_binance_data.sh custom 2023-01-01 2023-12-31 5m
   ```

3. **多周期数据**: 根据策略需求抓取多个周期
   ```bash
   ./fetch_binance_data.sh one-year 5m   # 短期策略
   ./fetch_binance_data.sh one-year 1h   # 中期趋势
   ./fetch_binance_data.sh one-year 1d   # 长期分析
   ```

4. **定期更新**: 定期抓取最新数据保持数据更新
   ```bash
   # 每周末抓取上周数据
   ./fetch_binance_data.sh recent 7 5m
   ```

## 📝 技术细节

- 币安合约API限制：**1200请求/分钟**

- 系统自动处理分批抓取
- 自动检测并跳过重复数据

### 性能优化

- 使用批量插入提高效率
- 数据库索引优化查询速度
- 并发控制避免资源竞争
