## 三、开单逻辑分析 🧠

### 典型交易模式识别

#### 模式1：连续加仓（常见）
```python
# 示例：2026-01-06 10:03-10:15
- 10:03:56 开空 0.01手 @4442.917 → 亏$7.96
- 10:03:58 开空 0.01手 @4442.904 → 亏$7.98
- 10:04:24 开空 0.01手 @4444.959 → 亏$5.92
- 10:04:26 开空 0.01手 @4444.966 → 亏$5.91
... (共19笔小单)
```

**特点**:
- 在同一分钟内连开多单
- 单笔仓位很小（0.01手）
- 方向一致，价格相近
- 明显是一个大仓位拆分成多个小单

**动机猜测**:
1. 降低单笔风险感知
2. 便于分批平仓
3. 心理暗示"不是重仓"
4. ⚠️ 本质上还是重仓！

#### 模式2：快速对冲（危险）
```python
# 示例：2026-01-07 03:19-03:20
03:19:38 买入 @4480.337 → 亏$0.75
03:19:38 卖出 @4480.203 → 平
03:19:38 买入 @4480.238 → 平
... (20秒内多空来回)
```

**特点**:
- 极短时间内多空互换
- 盈亏接近0
- 频繁开平仓

**问题**:
- ❌ 没有明确方向判断
- ❌ 可能是追单后发现错误再反向
- ❌ 情绪交易的典型表现

#### 模式3：扛单+爆仓（致命）
```python
# 示例：2026-02-02 多单爆仓
05:34:31 买入 0.01手 @4636.683 → 强平@4448.505 亏$188
05:40:52 买入 0.01手 @4612.743 → 强平@4448.505 亏$164
05:51:50 买入 0.01手 @4532.751 → 强平@4448.505 亏$84
06:22:37 买入 0.01手 @4492.295 → 强平@4448.505 亏$43
... 
最终：06:27:29 强平 0.2手 @4434.811 亏$803 💀
```

**灾难过程**:
1. 开多单后价格下跌
2. 不止损，反而不断加仓摊平成本
3. 从小单变大单(0.01→0.2)
4. 跌幅超过4%，触发强制平仓
5. 一次爆仓吞掉几十笔小盈利

- **无风险预算**: 不知道每笔最多能亏多少
