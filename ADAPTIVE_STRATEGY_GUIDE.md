# 自适应策略系统使用指南 - 2026年1月13日

## 🎯 系统概述

你的交易系统现在具备**自适应能力**，能够根据市场状态自动切换策略：

### 核心理念
```
震荡市 → 使用震荡市策略（均值回归）
趋势市 → 使用趋势市策略（趋势跟踪）
```

## 📊 市场状态识别（MarketRegimeDetector）

### 四种市场状态

| 市场状态 | ADX范围 | 波动率 | 适用策略 |
|---------|---------|--------|----------|
| **强趋势** | >35 | 高 | ✅ TrendFilter（EMA） |
| **弱趋势** | 25-35 | 中 | ✅ TrendFilter（EMA） |
| **盘整** | <20 | 低 | ✅ RangingMarket（布林带） |
| **震荡** | <20 | 高 | ✅ RangingMarket（布林带） |

### 判断依据

**MarketRegimeDetector** 综合以下3个指标：

1. **ADX（平均趋向指标）**
   - ADX > 35：强趋势
   - ADX 25-35：弱趋势  
   - ADX < 20：震荡/盘整

2. **波动率**（最近10根K线标准差/均价）
   - 高波动率 > 1.5%
   - 中波动率 1.0-1.5%
   - 低波动率 < 1.0%

3. **趋势一致性**（最近5根K线方向一致程度）
   - 高一致性 > 60%
   - 低一致性 < 60%

## 💡 策略详解

### 1️⃣ TrendFilter（趋势策略）- 权重12

**适用场景**: ADX ≥ 25的趋势市

**核心逻辑**:
- EMA20 > EMA50 + 价格 > EMA20 → 做多
- EMA20 < EMA50 + 价格 < EMA20 → 做空

**优化点**:
- ✅ 震荡市自动禁用（ADX<20）
- ✅ 高位过滤（价格接近最高点降低强度）
- ✅ 低位过滤（价格接近最低点降低强度）
- ✅ 权重降低（20→12），避免单一策略主导

**开仓条件**:
- 趋势强度 ≥ 15
- 价格不能过于极端
- 需要其他策略支持（权重12 < 阈值15）

---

### 2️⃣ RangingMarket（震荡市策略）- 权重15 ⭐新增⭐

**适用场景**: ADX < 20的震荡市/盘整市

**核心逻辑**:
```
做多信号（低买）:
价格位置 ≤ 20% (接近布林带下轨)
+ RSI < 35 (超卖)
+ Williams %R < -75 (超卖)
→ 得分 ≥ 8分 → 做多

做空信号（高卖）:
价格位置 ≥ 80% (接近布林带上轨)
+ RSI > 65 (超买)
+ Williams %R > -25 (超买)
→ 得分 ≥ 8分 → 做空
```

**评分系统**:
- RSI超卖/超买：5分
- Williams超卖/超买：4分
- 价格触及上/下轨：5分
- **总分≥8分才开仓**

**特点**:
- ✅ 只在震荡市启用
- ✅ 均值回归策略（不追涨杀跌）
- ✅ 快进快出（震荡市持仓时间短）
- ✅ 高确认度（需要≥8分）

## 🎓 策略哲学

### 核心思想

> **没有完美的策略，只有适合的策略。**
> 
> - 震荡市用震荡市的策略（均值回归）
> - 趋势市用趋势市的策略（趋势跟踪）
> - 自动识别，自动切换

### 为什么这样设计？

1. **EMA在震荡市失效**
   - 频繁金叉死叉 → 虚假信号
   - 追涨杀跌 → 高买低卖
   - **解决方案**：震荡市禁用EMA，改用布林带

2. **均值回归在趋势市危险**
   - 趋势中低买 → 持续下跌
   - 趋势中高卖 → 持续上涨
   - **解决方案**：趋势市禁用均值回归，改用趋势跟踪

3. **关键是识别市场状态**
   - ADX是最可靠的趋势强度指标
   - 结合波动率和一致性
   - 自动切换策略
