## 📝 数据库查询参考

### 查询信号强度vs胜率:
```sql
SELECT 
    CASE 
        WHEN signal_strength >= 90 THEN '90-100'
        WHEN signal_strength >= 70 THEN '70-89'
        WHEN signal_strength >= 50 THEN '50-69'
        ELSE '<50'
    END as strength_range,
    COUNT(*) as trade_count,
    SUM(CASE WHEN profit_loss > 0 THEN 1 ELSE 0 END) as win_count,
    ROUND(SUM(CASE WHEN profit_loss > 0 THEN 1 ELSE 0 END) * 100.0 / COUNT(*), 1) as win_rate,
    ROUND(SUM(profit_loss), 2) as total_pnl
FROM trade_order
WHERE DATE(create_time) = CURDATE()
AND close_time IS NOT NULL
GROUP BY strength_range
ORDER BY strength_range DESC;
```

### 查询过滤器统计:
```sql
-- 需要在代码中记录被过滤的信号(建议加个filter_log表)
SELECT 
    filter_reason,
    COUNT(*) as count
FROM signal_filter_log
WHERE DATE(create_time) = CURDATE()
GROUP BY filter_reason
ORDER BY count DESC;
```
