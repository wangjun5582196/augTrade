# 📊 获取Bybit历史K线数据指南

## 🎯 目标
获取2025年6月1日至2025年12月31日的黄金（XAUUSD）5分钟K线数据，用于回测。

## 📦 已创建的文件

1. ✅ `src/main/java/com/ltp/peter/augtrade/util/HistoricalDataFetcher.java` - 数据获取工具类
2. ✅ `src/main/java/com/ltp/peter/augtrade/controller/DataFetchController.java` - 数据获取接口

## 🚀 使用方法

### 方法1：通过API接口获取（推荐）

#### 步骤1：重新构建项目
在IntelliJ IDEA中：
```
Build → Rebuild Project
```

#### 步骤2：调用API获取数据

**快捷方式（获取2025年6月-12月数据）：**
```bash
curl -X POST "http://localhost:3131/api/data/fetch-2025-gold"
```

**自定义日期范围：**
```bash
curl -X POST "http://localhost:3131/api/data/fetch-historical" \
  -d "symbol=XAUUSD" \
  -d "interval=5" \
  -d "startDate=2025-06-01" \
  -d "endDate=2025-12-31"
```

**参数说明：**
- `symbol`: 交易对，如 XAUUSD（黄金）、BTCUSDT（比特币）
- `interval`: K线周期，如 1, 5, 15, 30, 60, 240, D（日线）
- `startDate`: 开始日期，格式 YYYY-MM-DD
- `endDate`: 结束日期，格式 YYYY-MM-DD

#### 步骤3：等待数据获取完成

数据获取过程中会显示进度：
```json
{
  "success": true,
  "message": "数据获取完成",
  "count": 52560,
  "symbol": "XAUUSD",
  "interval": "5",
  "startDate": "2025-06-01",
  "endDate": "2025-12-31"
}
```

### 方法2：使用Postman

1. 打开Postman
2. 创建POST请求：`http://localhost:3131/api/data/fetch-2025-gold`
3. 点击"Send"按钮
4. 查看响应结果

### 方法3：使用浏览器（需要额外工具）

由于是POST请求，需要使用浏览器插件（如Talend API Tester）或直接在浏览器控制台执行：

```javascript
fetch('http://localhost:3131/api/data/fetch-2025-gold', {
    method: 'POST'
})
.then(res => res.json())
.then(data => console.log(data));
```

## ⚠️ 注意事项

1. **重新构建项目**
   - 新创建的类需要重新编译才能被Spring Boot加载
   - 在IntelliJ中：`Build → Rebuild Project`

2. **数据重复问题**
   - 如果数据已存在，会自动跳过（不会重复插入）
   - 可以安全地多次执行

3. **时区问题**
   - 时间会自动转换为你的系统时区
   - 数据库中保存的是本地时间

4. **API限流**
   - 代码中已添加100ms延迟避免请求过快
   - 如遇到限流，会自动停止

5. **数据完整性**
   - 获取完成后，可以在数据库中查询验证：
   ```sql
   SELECT COUNT(*) FROM t_kline 
   WHERE symbol = 'XAUUSD' 
   AND `interval` = '5'
   AND open_time BETWEEN '2025-06-01' AND '2025-12-31';
   ```

## 🔍 验证数据

### 方法1：SQL查询
```sql
-- 查看数据总量
SELECT COUNT(*) as total FROM t_kline 
WHERE symbol = 'XAUUSD' AND `interval` = '5';

-- 查看时间范围
SELECT 
    MIN(open_time) as earliest,
    MAX(open_time) as latest,
    COUNT(*) as count
FROM t_kline 
WHERE symbol = 'XAUUSD' AND `interval` = '5';

-- 查看每天的数据量
SELECT 
    DATE(open_time) as date,
    COUNT(*) as count
FROM t_kline 
WHERE symbol = 'XAUUSD' 
AND `interval` = '5'
AND open_time BETWEEN '2025-06-01' AND '2025-12-31'
GROUP BY DATE(open_time)
ORDER BY date;
```

## 🎉 数据获取完成后

数据准备好后，就可以使用回测功能了：

1. 访问回测页面：`http://localhost:3131/api/backtest`
2. 配置回测参数：
   - 交易对：XAUUSD
   - K线周期：5m
   - 策略：SHORT_TERM / BREAKOUT / RSI
   - 时间范围：2025-06-01 至 2025-12-31
   - 初始资金：10000
3. 点击"开始回测"按钮
4. 查看回测结果和交易明细
