# 📊 回测策略说明

## 当前情况

### 问题
回测系统目前使用的是3个简化的测试策略：
- `SHORT_TERM` - 简单的均线+RSI+MACD组合
- `BREAKOUT` - 布林带突破
- `RSI` - 纯RSI超买超卖

### 你实际使用的策略
你的实盘交易使用的是 **StrategyOrchestrator** + **CompositeStrategy**，包含：
1. **BalancedAggressiveStrategy** - 平衡激进策略（权重7-10）
2. **BollingerBreakoutStrategy** - 布林带突破策略（权重6）
3. **RSIStrategy** - RSI策略（权重8-9）
4. **WilliamsStrategy** - Williams %R策略（权重8）

这是一个更复杂、更智能的多策略组合系统。

```java
private BacktestResult executeCompositeBacktest(String backtestId, String symbol, 
                                                 String interval, List<Kline> klines, 
                                                 BigDecimal initialCapital) {
    log.info("执行组合策略回测（使用真实StrategyOrchestrator）");
    
    BigDecimal capital = initialCapital;
    BigDecimal maxCapital = initialCapital;
    List<BacktestTrade> trades = new ArrayList<>();
    BacktestTrade openPosition = null;
    
    for (int i = 100; i < klines.size(); i++) {
        // 准备历史数据（最新的在最前面，符合StrategyOrchestrator的要求）
        List<Kline> historicalData = new ArrayList<>(klines.subList(0, i + 1));
        Collections.reverse(historicalData);  // 反转顺序
        
        Kline currentKline = klines.get(i);
        BigDecimal currentPrice = currentKline.getClosePrice();
        
        // 检查止盈止损...（同现有逻辑）
        
        // 如果没有持仓，使用真实策略生成信号
        if (openPosition == null) {
            MarketContext context = MarketContext.builder()
                .symbol(symbol)
                .klines(historicalData)
                .currentPrice(currentPrice)
                .timestamp(currentKline.getTimestamp().toInstant(ZoneOffset.UTC).toEpochMilli())
                .build();
            
            // 计算所有指标
            strategyOrchestrator.calculateAllIndicators(context);
            
            // 使用CompositeStrategy生成信号
            TradingSignal signal = compositeStrategy.generateSignal(context);
            
            if (signal != null && signal.getStrength() >= 70) {  // 与实盘一致的阈值
                String side = signal.getType() == TradingSignal.SignalType.BUY ? "BUY" : "SELL";
                openPosition = openPosition(backtestId, symbol, side, currentPrice, 
                                           currentKline.getTimestamp(), capital);
                openPosition.setSignalDescription(signal.getReason());
            }
        }
    }
    
    // 保存交易记录和计算结果...
    return calculateBacktestResult(symbol, interval, initialCapital, capital, maxCapital, trades);
}
```
