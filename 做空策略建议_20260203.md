## 三、风险控制要点

### 1. 仓位管理
```
激进型: 账户资金的30-40%
稳健型: 账户资金的20-30%
保守型: 账户资金的10-20%
```

### 2. 止损纪律
- ⚠️ **严格止损**: 必须设置止损,不能抗单!
- 建议止损幅度: 1.2-2.0% (根据入场点位)
- 如果价格突破止损价,立即平仓,不要幻想回调

### 3. 止盈策略
```
分批止盈:
- 30%仓位: 到达T1立即止盈,锁定利润
- 40%仓位: 到达T2止盈,保证盈利
- 30%仓位: 追求T3,放飞利润
```

### 4. 移动止损
```
当盈利达到2%时:
→ 将止损移至成本价(保本)

当盈利达到4%时:
→ 将止损移至成本价+2%(锁定2%利润)
```

### 5. 时间止损
- 如果持仓3天价格没有明显下跌,考虑平仓
- 避免在横盘震荡中消耗时间成本
