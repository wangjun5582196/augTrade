## 📊 支持的策略

### 1. SHORT_TERM - 短线趋势策略
- **指标组合**：SMA(5,10,20) + RSI(14) + MACD
- **适用场景**：趋势明显的市场
- **特点**：快速进出，捕捉短期趋势

### 2. BREAKOUT - 布林带突破策略
- **指标组合**：布林带(20,2)
- **适用场景**：震荡后突破
- **特点**：突破上轨做多，突破下轨做空

### 3. RSI - RSI超买超卖策略
- **指标组合**：RSI(14)
- **适用场景**：震荡市
- **特点**：超买区域做空，超卖区域做多

## 🎯 回测参数说明

### 止盈止损设置
- **止盈**：2%（开仓价的2%）
- **止损**：1%（开仓价的1%）
- **手续费**：0.05%（双边，开仓+平仓）

### 数据要求
- 最少需要 **50根K线** 才能计算技术指标
- 建议使用 **1000根以上** K线数据进行回测
- 时间范围越长，回测结果越准确

---

## 📈 回测指标说明

| 指标 | 说明 | 参考值 |
|------|------|--------|
| 收益率 | (最终资金-初始资金)/初始资金×100% | >0% 为盈利 |
| 胜率 | 盈利交易次数/总交易次数×100% | >50% 较好 |
| 盈亏比 | 平均盈利/平均亏损 | >2 优秀，>1.5 良好 |
| 夏普比率 | 风险调整后的收益 | >1 良好，>2 优秀 |
| 最大回撤率 | 从峰值到谷底的最大跌幅 | <20% 较好 |

## 🔍 验证数据

### 检查K线数据是否获取成功
```sql
SELECT 
    symbol,
    `interval`,
    MIN(timestamp) as earliest,
    MAX(timestamp) as latest,
    COUNT(*) as count
FROM t_kline 
WHERE symbol = 'XAUUSD' 
AND `interval` = '5'
GROUP BY symbol, `interval`;
```

### 查看回测结果
```sql
SELECT 
    backtest_id,
    strategy_name,
    initial_capital,
    final_capital,
    return_rate,
    win_rate,
    total_trades,
    status,
    create_time
FROM t_backtest_result 
ORDER BY create_time DESC 
LIMIT 10;
```

## ⚠️ 常见问题

### Q1: 页面显示404
**A**: 需要先重新构建项目，让新的Controller被编译和加载。

### Q2: 回测时提示"数据不足"
**A**: 确保数据库中有足够的历史K线数据（至少50根）。

### Q3: 数据获取很慢
**A**: 这是正常的，Bybit API有速率限制，代码已添加100ms延迟。

### Q4: 提示"Bad Request"错误
**A**: 确保项目已重新构建，新的Controller代码已生效。

### Q5: interval字段报SQL错误
**A**: 已添加反引号转义，确保使用最新的Entity代码。
