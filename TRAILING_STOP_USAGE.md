## 🎯 针对您的情况（当前盈利$41）

您现在有一笔交易，**目标盈利$100，当前盈利$41**。移动止损功能已经实现并配置好，可以帮助您：

---

## ⚙️ 配置说明

在 `application.yml` 中已添加移动止损配置：

```yaml
trading:
  risk:
    # 移动止损配置
    trailing-stop:
      enabled: true              # ✅ 启用移动止损
      trigger-profit: 30.0       # 触发阈值：盈利达到$30时启动
      distance: 10.0             # 跟踪距离：$10
      lock-profit-percent: 70.0  # 锁定比例：首次触发时锁定70%利润
```

### 参数说明

| 参数 | 说明 | 当前值 | 建议调整 |
|------|------|--------|----------|
| `enabled` | 是否启用移动止损 | `true` | 保持启用 |
| `trigger-profit` | 触发阈值（美元） | `30.0` | 当前$41已触发 |
| `distance` | 跟踪距离（美元） | `10.0` | 可调整为8-12 |
| `lock-profit-percent` | 首次锁定比例（%） | `70.0` | 锁定70%利润 |

## 🎓 核心要点

### ✅ 移动止损的优势

1. **自动保护**：价格上涨时自动上调止损
2. **趋势跟踪**：不限制上涨空间
3. **利润锁定**：回调时保护已有收益
4. **无需盯盘**：系统自动执行

### ⚠️ 注意事项

1. **止损只能上调**：多头止损价只会上升，永不下降
2. **触发后不可逆**：一旦启用移动止损，无法切回固定止损
3. **市场波动**：剧烈震荡可能触发止损
4. **跟踪距离**：太紧容易被震出，太松保护不足

**当前状态**：✅ 已满足触发条件（$41 > $30）

**系统行为**：

- 初始锁定利润：$41 * 70% = $28.7
- 止损价设置为：入场价 + $28.7
