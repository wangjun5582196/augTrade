## 🎯 二、推荐补充的实时性指标

### 2.1 极高实时性指标（推荐⭐⭐⭐）

#### 1. **价格动量指标 (Price Momentum)** ⭐⭐⭐⭐⭐
**实时性**: 极高（1-5根K线）  
**计算公式**: `Momentum = 当前价 - N周期前价格`  
**优势**:
- 直接反映价格变化速度，无滞后
- 可以用2根、3根、5根K线计算不同周期动量
- 适合捕捉快速突破和反转

**应用场景**:
```java
// 短期动量（2根K线）
BigDecimal momentum2 = currentPrice - price2;
if (momentum2 > ATR * 0.3) → 看涨信号

// 中期动量（5根K线）
BigDecimal momentum5 = currentPrice - price5;
if (momentum5 > ATR * 0.5 && momentum2 > 0) → 强烈看涨
```

**建议权重**: 3-5分（在综合策略中）

---

#### 2. **价格变化率 (Rate of Change, ROC)** ⭐⭐⭐⭐⭐
**实时性**: 极高（可自定义周期）  
**计算公式**: `ROC = ((当前价 - N周期前价格) / N周期前价格) × 100`  
**优势**:
- 百分比表示，便于不同品种对比
- 比动量指标更标准化
- 可以设置阈值（如ROC > 0.5%做多）

**应用场景**:
```java
// 3根K线的价格变化率
double roc3 = (currentPrice - price3) / price3 * 100;

// 信号规则
if (roc3 > 0.3 && roc3 < 2.0) → 健康上涨，可做多
if (roc3 > 2.0) → 暴力行情，暂停交易（你已有此逻辑）
```

**建议权重**: 3-4分

---

#### 3. **成交量突破指标 (Volume Breakout)** ⭐⭐⭐⭐⭐
**实时性**: 极高（当前K线）  
**计算公式**: `Volume Ratio = 当前成交量 / 20周期平均成交量`  
**优势**:
- 成交量是**领先指标**，先于价格变化
- 突放量往往预示大行情启动
- 你当前的OBV是累计指标，不够实时

**应用场景**:
```java
// 成交量放大倍数
double volumeRatio = currentVolume / avgVolume20;

// 信号规则
if (volumeRatio > 1.5 && price突破关键位) → 确认突破有效，权重+5
if (volumeRatio < 0.5) → 假突破概率高，降低权重
```

**建议权重**: 4-6分（成交量确认非常重要）

---

### 2.2 中高实时性指标（推荐⭐⭐）

#### 4. **Keltner通道 (Keltner Channels)** ⭐⭐⭐⭐
**实时性**: 中高（基于EMA20+ATR）  
**计算公式**: 
- 中轨 = EMA20
- 上轨 = EMA20 + ATR × 2
- 下轨 = EMA20 - ATR × 2

**优势**:
- **比布林带更实时**：使用EMA而非SMA，使用ATR适应波动
- 你当前的布林带用SMA20，Keltner用EMA20更快
- 在趋势市场中表现更好

**应用场景**:
```java
// 替代或补充布林带
if (price > keltnerUpper && ADX > 25) → 强势突破，做多
if (price < keltnerLower && ADX > 25) → 强势突破，做空

// 价格回归中轨时入场（高胜率）
if (price触及上轨后回落到中轨附近 && ADX > 20) → 回调买入
```

**建议权重**: 5-6分（可替代或补充布林带）

---

#### 5. **Donchian通道突破 (Donchian Channels)** ⭐⭐⭐⭐
**实时性**: 高（基于N周期高低点）  
**计算公式**:
- 上轨 = N周期最高价
- 下轨 = N周期最低价
- 中轨 = (上轨 + 下轨) / 2

**优势**:
- **海龟交易法则核心指标**，经过验证的突破系统
- 比布林带更简单直接，无统计假设
- 适合趋势跟随策略

**应用场景**:
```java
// 20周期Donchian通道
double upper = max(klines, 20);
double lower = min(klines, 20);

// 信号规则
if (price突破20周期高点 && ADX > 25) → 强烈做多信号
if (price跌破20周期低点 && ADX > 25) → 强烈做空信号
```

**建议权重**: 6-7分（突破系统的核心）

---

#### 6. **HMA (Hull Moving Average, 赫尔移动平均)** ⭐⭐⭐⭐
**实时性**: 高（比EMA更快）  
**计算公式**: `HMA = WMA(2×WMA(n/2) - WMA(n), sqrt(n))`  
**优势**:
- **几乎无滞后的移动平均线**
- 比EMA快50%以上
- 保持平滑性的同时提高响应速度

**应用场景**:
```java
// 替代EMA20作为趋势判断
double hma20 = calculateHMA(klines, 20);

// 信号规则
if (price > hma20 && hma20向上 && ADX > 20) → 确认上涨趋势
if (hma20由下转上) → 趋势反转信号（比EMA金叉早2-3根K线）
```

**建议权重**: 4-5分

---

### 2.3 特殊实时性指标（推荐⭐）

#### 7. **摆动高低点 (Swing High/Low)** ⭐⭐⭐⭐⭐
**实时性**: 极高（3-5根K线确认）  
**定义**:
- 摆动高点：中间K线高点 > 左右两边K线高点
- 摆动低点：中间K线低点 < 左右两边K线低点

**优势**:
- **支撑阻力位的实时识别**
- 突破摆动高点 = 确认上涨，跌破摆动低点 = 确认下跌
- 你当前的价格位置过滤可以改用摆动点

**应用场景**:
```java
// 识别最近的摆动高点和低点
double lastSwingHigh = findLastSwingHigh(klines, 5);
double lastSwingLow = findLastSwingLow(klines, 5);

// 信号规则
if (price突破lastSwingHigh && 成交量放大) → 突破确认，做多+6分
if (price在lastSwingLow上方反弹) → 支撑有效，做多+4分
```

**建议权重**: 6-8分（非常重要的价格行为指标）

---

#### 8. **市场结构分析 (Market Structure)** ⭐⭐⭐⭐⭐
**实时性**: 极高（价格行为）  
**定义**:
- 上涨结构：连续的更高高点 + 更高低点
- 下跌结构：连续的更低高点 + 更低低点
- 结构破坏：趋势反转的早期信号

**优势**:
- **最原始的价格行为分析**，无滞后
- 可以在趋势反转早期发现信号
- 专业交易员必看的指标

**应用场景**:
```java
// 判断市场结构
if (连续3个更高高点 && 连续3个更高低点) → 确认上涨结构
if (上涨结构中出现更低低点) → 结构破坏，考虑平仓或反手

// 入场规则
if (上涨结构 && price回调到前一个更高低点附近) → 高胜率做多机会
```

**建议权重**: 8-10分（顶级信号）

```java
// 在IndicatorService中添加
public double calculateVolumeRatio(List<Kline> klines, int period) {
    if (klines == null || klines.size() < period + 1) {
        return 1.0;
    }
    
    BigDecimal currentVolume = klines.get(0).getVolume();
    
    BigDecimal sum = BigDecimal.ZERO;
    for (int i = 1; i <= period; i++) {
        sum = sum.add(klines.get(i).getVolume());
    }
    
    BigDecimal avgVolume = sum.divide(BigDecimal.valueOf(period), 4, RoundingMode.HALF_UP);
    
    if (avgVolume.compareTo(BigDecimal.ZERO) == 0) {
        return 1.0;
    }
    
    return currentVolume.divide(avgVolume, 4, RoundingMode.HALF_UP).doubleValue();
}

// 在CompositeStrategy中使用
double volumeRatio = indicatorService.calculateVolumeRatio(context.getKlines(), 20);

// 成交量评分（0-5分）
int volumeScore = 0;
if (volumeRatio > 1.5) {
    volumeScore += 5;  // 强放量
    log.info("[{}] 📊 成交量突破 ({}倍)，+5分", STRATEGY_NAME, String.format("%.2f", volumeRatio));
} else if (volumeRatio > 1.2) {
    volumeScore += 3;  // 温和放量
} else if (volumeRatio < 0.5) {
    volumeScore -= 2;  // 缩量，降低权重
    log.warn("[{}] ⚠️ 成交量萎缩 ({}倍)，-2分", STRATEGY_NAME, String.format("%.2f", volumeRatio));
}

buyScore += volumeScore;
```

#### 3. 摆动高低点指标
```java
// 创建新文件 SwingPointCalculator.java
@Service
public class SwingPointCalculator {
    
    public static class SwingPoint {
        private BigDecimal price;
        private int index;
        private boolean isHigh;  // true=摆动高点, false=摆动低点
        
        // getters...
    }
    
    /**
     * 查找最近的摆动高点
     * @param klines K线数据
     * @param lookback 回看周期（通常3-5）
     */
    public SwingPoint findLastSwingHigh(List<Kline> klines, int lookback) {
        if (klines.size() < lookback * 2 + 1) {
            return null;
        }
        
        for (int i = lookback; i < klines.size() - lookback; i++) {
            Kline current = klines.get(i);
            boolean isSwingHigh = true;
            
            // 检查左右两边
            for (int j = 1; j <= lookback; j++) {
                if (current.getHighPrice().compareTo(klines.get(i - j).getHighPrice()) <= 0 ||
                    current.getHighPrice().compareTo(klines.get(i + j).getHighPrice()) <= 0) {
                    isSwingHigh = false;
                    break;
                }
            }
            
            if (isSwingHigh) {
                SwingPoint sp = new SwingPoint();
                sp.price = current.getHighPrice();
                sp.index = i;
                sp.isHigh = true;
                return sp;
            }
        }
        
        return null;
    }
    
    public SwingPoint findLastSwingLow(List<Kline> klines, int lookback) {
        // 类似逻辑，查找摆动低点
    }
}

// 在CompositeStrategy中使用
SwingPoint lastSwingHigh = swingPointCalculator.findLastSwingHigh(context.getKlines(), 5);
SwingPoint lastSwingLow = swingPointCalculator.findLastSwingLow(context.getKlines(), 5);

BigDecimal currentPrice = context.getCurrentPrice();

// 摆动点评分（0-7分）
int swingScore = 0;

if (lastSwingHigh != null) {
    // 突破摆动高点
    if (currentPrice.compareTo(lastSwingHigh.getPrice()) > 0) {
        swingScore += 7;
        log.info("[{}] 🚀 突破摆动高点 ${} → +7分", 
                STRATEGY_NAME, lastSwingHigh.getPrice());
    }
}

if (lastSwingLow != null) {
    // 价格在摆动低点上方，支撑有效
    BigDecimal distance = currentPrice.subtract(lastSwingLow.getPrice());
    if (distance.doubleValue() > 0 && distance.doubleValue() < atr * 0.5) {
        swingScore += 4;
        log.info("[{}] ✅ 支撑位有效 (摆动低点${}附近) → +4分", 
                STRATEGY_NAME, lastSwingLow.getPrice());
    }
}

buyScore += swingScore;
```
