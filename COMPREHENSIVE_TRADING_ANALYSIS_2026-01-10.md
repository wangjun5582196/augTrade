- 止盈$30从未触达

- 平均盈利只有$11.69

- 信号反转会在到达止盈前提前平仓

**建议方案** 🎯:

```yaml
bybit.risk.trailing-stop.distance: 8.0         # 跟踪距离$8
bybit.risk.trailing-stop.lock-profit-percent: 60.0  # 锁定60%利润
```

**预期效果**:
- 方案A：止盈触达率从0%提升至15-25%
- 方案B：在不改变止盈目标的情况下，锁定更多利润

```java
public class MLPredictionService {
    private List<PredictionResult> predictionHistory = new ArrayList<>();
    
    public double getRecentAccuracy() {
        // 统计最近50次预测的准确率
        return predictionHistory.stream()
            .limit(50)
            .filter(p -> p.isPredictionCorrect())
            .count() / 50.0;

    // 根据准确率动态调整ML权重
    public int getMLScore() {

    }
}
```

```java
public class MoneyManagementService {
    
    public int calculatePositionSize(BigDecimal accountBalance, 
                                     double recentWinRate,
                                     int consecutiveLosses) {
        // 基础仓位：账户余额的2%
        int baseSize = accountBalance.multiply(new BigDecimal("0.02"))
            .divide(new BigDecimal("4500"), 0, RoundingMode.DOWN)  // 假设金价4500
            .intValue();
        
        // 根据胜率调整
        if (recentWinRate > 0.65) {
            baseSize = (int)(baseSize * 1.5);  // 高胜率时增加50%
        } else if (recentWinRate < 0.45) {
            baseSize = (int)(baseSize * 0.5);  // 低胜率时减少50%
        }
        
        // 连续亏损保护
        if (consecutiveLosses >= 3) {
            baseSize = Math.max(baseSize / 2, 5);  // 至少保持5手
        }
        
        // 限制范围
        return Math.min(Math.max(baseSize, 5), 20);  // 5-20手之间
    }
}
```
