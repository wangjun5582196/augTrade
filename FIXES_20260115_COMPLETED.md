## 五、核心修复逻辑

### 修复策略

#### 1. 三层过滤机制
```
第一层: ADX市场状态过滤
├─ ADX < 15: 完全禁止 ❌
├─ ADX 15-20: 需要9分以上 ⚠️
└─ ADX > 20: 正常交易 ✅

第二层: 趋势方向过滤
├─ 下降趋势(+DI < -DI): 禁止做多 ❌
├─ 上升趋势(+DI > -DI): 做空+3分门槛 ⚠️
└─ 中性趋势: 正常评分 ✅

第三层: Williams R严格过滤
├─ 做多: 必须< -60 (中性区-40~-60不给分)
└─ 做空: 必须> -40
```

#### 2. 信号强度动态阈值
```
震荡市(ADX<20): 需要9分 (原7分)
弱趋势(ADX 20-30): 需要7分
强趋势(ADX>30): 需要7分
  - 上升趋势: 做多自动+3分
  - 下降趋势: 做空自动+3分
```

#### 3. 信号反转严格控制
```
禁止反转条件(满足任一即禁止):
├─ 盈利 > $50
├─ 盈亏 在 ±$20内
├─ 移动止损已启动
├─ 信号强度 < 90
└─ 亏损 < $30

允许反转条件(必须全部满足):
├─ 亏损 > $30 ✅
├─ 信号强度 ≥ 90 ✅
├─ 持仓时间 > 保护期 ✅
└─ 未启动移动止损 ✅
```
