## 📝 二、新策略特点

### 2.1 SimplifiedTrendStrategy核心逻辑

```
步骤1: 计算核心指标
  - ADX: 判断趋势强度
  - ATR: 计算波动率
  - EMA20/50: 判断趋势方向
  - 当前价格与动量

步骤2: 三重市场环境过滤
  ⛔ 过滤1: ADX < 20 → HOLD（震荡市）
  ⛔ 过滤2: ATR > 6.0 → HOLD（高波动）
  ⛔ 过滤3: 15分钟涨跌 > 1% → HOLD（暴力行情）

步骤3: 强趋势顺势交易
  ADX 20-25: 弱趋势，等待更好机会
  ADX > 25: 强趋势，开始信号判断
  
  买入条件:
    - EMA20 > EMA50（上涨趋势）
    - 价格回调到EMA20附近（-0.5% ~ +0.3%）
    - 动量向上（确认）
    
  卖出条件:
    - EMA20 < EMA50（下跌趋势）
    - 价格反弹到EMA20附近（-0.3% ~ +0.5%）
    - 动量向下（确认）
```

### 2.2 止损止盈设置

```
止损: ATR * 3.0
止盈: ATR * 4.0
盈亏比: 1.33:1
```
